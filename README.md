**A comprehensive cross-platform fitness and nutrition tracking application built with Flutter.**

## Description

Fitia is a production-ready Flutter application for fitness coaching, nutrition tracking, and workout management. Designed with a custom theme system and responsive UI that works seamlessly across mobile, web, and desktop platforms.

## Features

- **Personalized Coaching** - Custom fitness coaching programs
- **Nutrition Tracking** - Meal logging and macro tracking (offline capable)
- **Workout Management** - Exercise routines and progress tracking
- **Shopping Integration** - Fitness product catalog
- **Custom Theme System** - Multiple theme variants including "Panda Rojo" custom design
- **Cross-Platform** - Single codebase runs on Android, iOS, Web, Windows, macOS, and Linux
- **Offline-First** - Core functionality works without internet connection

## Tech Stack

- **Flutter 3.x** - UI framework
- **Dart** - Programming language
- **Riverpod** - State management
- **Custom Theme Engine** - Flexible theming with multiple variants
- **Responsive Design** - Adapts to any screen size

## Getting Started

### Prerequisites
- Flutter SDK 3.x or higher
- Dart SDK
- Android Studio / Xcode (for mobile builds)

### Installation
```bash
git clone https://github.com/Eltocomohco/fitia.git
cd fitia
flutter pub get
flutter run
```

### Build for Production
```bash
# Android
flutter build apk --release

# iOS
flutter build ios --release

# Web
flutter build web --release

# Desktop (Windows/macOS/Linux)
flutter build <platform>
```
